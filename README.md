
# CodeAlpha C++ Internship Tasks

This repository contains 3 C++ projects completed as part of the CodeAlpha Internship.

## Tasks Implemented

### 1. CGPA Calculator
- Calculates CGPA based on grades and credit hours.
- Uses basic input/output and arithmetic logic.

### 2. Login & Registration System
- Allows user registration and login.
- Stores credentials in files.
- Validates login using stored data.

### 3. Sudoku Solver
- Solves a 9x9 Sudoku puzzle using backtracking.
- Ensures all Sudoku constraints are followed.
